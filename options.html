Instructions

Disable "Ask where to save each file before downloading" and select a download directory with enough space.

We will download the files one by one.

You also need to allow the website to download multiple files ("... automatic downloads of multiple files"). It will ask after the second automatic download or so for that.

Unfortunately, every 10mins or so, Google will request to reenter your Google password. To automate this, you need to store the Google password below. It is stored in the local storage in plain text!

You could run another script in the background which frequently checks for new files, and further processes them. See the backup-google-takeout.py script.
